# AGENTS.md

このファイルは、このリポジトリで作業するコーディングエージェント向けの注意事項です。
一般的な使い方やセットアップ手順は `README.md` を参照してください。

## ルール

1. **秘密情報をコミットしない**
   - トークン、秘密鍵、認証情報、ホスト固有設定は追加しないでください。
   - 機密情報が必要な変更は `../secure-dotfiles` で扱ってください。

2. **XDG Base Directory を優先する**
   - XDG で表現できるものは、可能な限り `home/` ではなく `config/` 配下で管理してください。
   - `config/` は `XDG_CONFIG_HOME`（通常は `~/.config`）向けです。

3. **定義を同期する**
   - bootstrap の挙動を変えたら、`README.md` も更新してください。

4. **変更は小さく保つ**
   - 既存のスタイルを維持し、無関係なリフォーマットは避けてください。

5. **macOS 前提で考える**
   - 明確なガードがない OS 依存変更は避けてください。

## 検証

必要に応じて以下を確認してください。

```sh
bash -n scripts/bootstrap.sh
```
